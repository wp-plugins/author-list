== Description ==
Display the list of authors with gravatar image and show the total number of post count on hover

Through this plugin user can,

* Able to show the Author list through widget.
* Onhover author image, the total number of post, user name and gravatar image will be displayed.
* Admin can set the limit of authors to display.
* Admin can set the gravatar image size to be 24x24, 48x48 or 96x96 pixel.
* Admin can customize the widget title name.
* Authors will be sort based on the number of post they published.

For More information : http://buffercode.com/author-list-wordpress-plugin/

v 1.1

* Authors will be sort based on the number of post they published.

v 1.0

* Public release

== Installation ==
1. Upload the "author-list" directory to the plugins directory.
2. Go to the plugins setting page and activate "Author List"
3. Go to Appearance --> Widget --> Author List Widget to appropriate location to display the Author List.
4. Enter the customize title to display in the Author List

For More information : http://buffercode.com/author-list-wordpress-plugin/
